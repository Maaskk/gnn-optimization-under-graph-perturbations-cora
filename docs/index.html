Research protocol

A fixed comparison protocol for every optimizer.

Same Cora split, same GCN architecture, same seed, same epochs, same learning rate. The comparison isolates optimizer behavior.

  • Weight decay 0.0005
  • Dropout 0.5
  • 200 epochs
