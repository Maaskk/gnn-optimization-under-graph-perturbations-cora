Protocole de recherche

Un protocole fixe pour comparer chaque optimiseur.

Même split Cora, même architecture GCN, même graine aléatoire, mêmes époques, même taux d'apprentissage. La comparaison isole le comportement de l'optimiseur.

Modèle

  • GCN à deux couches
  • 1433 attributs d'entrée
  • 16 canaux cachés
  • 7 classes en sortie

Optimisation

  • Taux d'apprentissage 0,01
  • Décroissance des poids 0,0005
  • Dropout 0,5
  • 200 époques

Évaluation

  • Précision test
  • Macro F1-score
  • Perte finale d'entraînement
  • Temps d'entraînement
